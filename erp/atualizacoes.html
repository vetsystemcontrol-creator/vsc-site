Backup & Atualizações (Padrão Canônico)

Local fixo de pacotes (sempre):

C:\Vet System Control - Equine\updates\inbox\<PACOTE>\

Para abrir o painel de aplicação:

  1. Execute: C:\Vet System Control - Equine\updater\RUN_UPDATER.cmd
  2. Abra: http://127.0.0.1:8099
Abrir Painel de Atualizações

Logs (fixo):

C:\Vet System Control - Equine\updater\logs\

Backups (fixo):

C:\Vet System Control - Equine\Vet_Backups\

Observação: se o painel não abrir, é porque o serviço do updater não está em execução. Inicie pelo arquivo RUN_UPDATER.cmd e tente novamente.
