FAQ

Respostas curtas — para não gerar dúvida no primeiro contato.

O acesso é pelo domínio principal?

O domínio principal é o site institucional. O sistema roda no subdomínio do app. Isso é padrão SaaS e evita problemas de manutenção.

Posso salvar esse site no Cloudflare Pages?

Sim. Este pacote já está pronto para Cloudflare Pages (site estático, leve e rápido).

Tem SEO?

Sim: title/description, Open Graph, canonical, sitemap e robots, além de performance (Core Web Vitals) com JS mínimo.
