# Vet System Control | Equine — Site Institucional

## ATENÇÃO: este é o repositório do SITE PÚBLICO

Este repositório é o responsável pelo site institucional publicado em:

- https://www.vetsystemcontrol.com.br
- https://vsc-site.pages.dev

Projeto Cloudflare Pages:

- vsc-site

## O que alterar aqui

Use este repositório para alterações em:

- landing page
- home do site
- copy institucional
- SEO
- header
- hero
- seções comerciais
- CTAs
- estilos do site público
- imagens institucionais
- textos de demonstração no site

## O que NÃO alterar aqui

Não usar este repositório para alterar:

- lógica interna do ERP
- módulos reais do sistema
- regras de negócio
- banco de dados
- demo interativa do ERP
- Workers da aplicação
- API
- sincronização real
- autenticação do ERP

## Repositório da demo / ERP

As alterações do ambiente demonstrativo e do ERP demo devem ser feitas no repositório/projeto correspondente à demo, não aqui.

Referência:

- Site institucional: vsc-site
- Demo / ERP demo: vsc-erp-demo

## Regra obrigatória antes de qualquer alteração

Antes de editar, confirmar:

1. A alteração é no site público?
   - Sim: usar este repositório.
   - Não: procurar o repositório da demo/ERP.

2. O domínio afetado é www.vetsystemcontrol.com.br?
   - Sim: usar este repositório.

3. O objetivo é alterar módulos reais do ERP?
   - Sim: não usar este repositório.

## Deploy

Este repositório publica automaticamente no Cloudflare Pages pelo projeto:

- vsc-site

Após push na branch main, conferir o deploy no painel do Cloudflare Pages.
